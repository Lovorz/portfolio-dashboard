Value Screen #1: Price-to-Book Value (P/BV)
P/BV คืออะไร?
P/BV = รำคำหุ ้ น÷ มูลค่ำทำงบัญชีต่อหุ ้ น(Book Value per Share)
ถ ้ ำP/BV < 1 หมำยควำมว่ำ ตลำดให ้มูลค่ำบริษัทต ่ำกว่ำสินทร ัพย ์ทำงบัญชี
หลักฐำนที่น่ำสนใจ
• งำนวิจัยระยะยำว: หุ ้ นP/BV ต ่ำให ้ผลตอบแทนสูงกว่ำ P/BV 
สูง
• ได ้ผลทั้งในตลำดสหร ัฐฯ และตลำดต่ำงประเทศ
• Fama-French Three Factor Model ยืนยันว่ำ P/BV เป ็ น
ปัจจัยส ำคัญ
ข้อระวัง (Caveat Emptor)
• P/BV ต ่ำอำจ = ควำมเสี่ยงสูง (บริษัท ใกล ้ ล ้ มละลำย)
• บริษัทที่ ROE < Cost of Equity ควรซื้อขำยต ่ำกว่ำ Book 
Value เสมอ
• ไม่ใช่ทุกหุ ้ นP/BV ต ่ำที่ถูกมูลค่ำ บำงตัวถูกเพรำะ 'แย่จริงๆ'
สูตรคิด P/BV ส ำหร ับบริษัทที่เติบโตสม ่ำเสมอ:
P/BV = (ROE − g) ÷ (ke − g)  โดย g = stable growth rate, ke = cost of equity

Value Screen #2: Price-to-Earnings (P/E Ratio)
P/E = รำคำหุ้น ÷ ก ำไรต่อหุ้น(EPS) | ตีควำม: ตลำดยอมจ่ำย X บำทต่อก ำไร1 บำท
หลักฐำน
• หุ ้ นP/E ต ่ำมักให ้ผลตอบแทนสูงกว่ำหุ ้น P/E สูง ในระยะยำว
• ผลนี้พบในทุกช่วงเวลำ ทุกตลำดทั่วโลก
• เป ็ นScreen หลักของ Ben Graham
กลไกที่ท ำให้ได้ผล
• P/E ต ่ำ = ตลำดคำดหวังต ่ำ → ถ ้ ำบริษัทท ำได ้ ดีกว่ำคำด รำคำ
พุ่ง
• หุ ้ นP/E ต ่ำจ ำนวนมำกอยู่ในธุรกิจ Mature หรือธุรกิจที่ตลำด
คำด Growth ต ่ำ
สิ่งที่อำจผิดพลำด (What Can Go Wrong?):
• 1️⃣ก ำไรมีควำมเสี่ยงสูง (Risky Earnings): P/E ต ่ำ อำจเป ็ นเพรำะนักบัญชีท ำตัวเลขดูดี(Earnings Manipulation) ไม่ใช่แก่นแท ้
• 2️⃣ภำษี (Tax Cost): หุ ้ นP/E ต ่ำมักจ่ำยปันผลสูง → ถูกหักภำษีปันผลมำกกว่ำ
• 3️⃣กำรเติบโตต ่ำ (Low Growth): P/E ต ่ำ อำจบอกว่ำธุรกิจ ไม่มีอนำคตเติบ โต ไม่ได ้ แล ้ ว
